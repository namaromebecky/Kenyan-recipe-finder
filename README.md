🍛 Kenyan Recipe Finder

A beautiful React application for discovering authentic Kenyan recipes. Built as a school project with modern web technologies.

## 🎨 Features

- **Authentic Kenyan Recipes**: 5 researched dishes with Swahili names
- **Beautiful UI**: Modern design with Kenyan-inspired colors (green, orange, amber)
- **Search Functionality**: Find recipes by name, ingredients, or category
- **Detailed Views**: Step-by-step instructions with ingredient quantities
- **Cultural Information**: Learn about Kenyan food heritage
- **Responsive Design**: Works perfectly on mobile, tablet, and desktop
- **School Project Ready**: Complete with documentation and clean code

## 🍽️ Featured Recipes

1. **Nyama Choma** - Traditional Kenyan grilled meat
2. **Ugali with Sukuma Wiki** - Staple maize porridge with collard greens
3. **Chapati** - Soft, layered flatbread
4. **Mandazi** - Swahili coconut doughnuts
5. **Kachumbari** - Fresh tomato and onion salad

## 🛠️ Technologies Used

- **React 18** - Frontend framework
- **Vite** - Build tool and development server
- **Tailwind CSS** - Utility-first CSS framework
- **Local Data** - No API required, all recipes included
- **Git & GitHub** - Version control and deployment

## 🚀 Getting Started

### Prerequisites
- Node.js (v14 or higher)
